# safebank
O SafeBank é uma aplicação web voltada para dispositivos móveis que permite aos usuários registrar, consultar e compartilhar relatos de golpes bancários, ajudando outras pessoas a reconhecer e evitar fraudes financeiras.
O projeto reproduz a experiência visual de um aplicativo mobile estilo Rede Social, com interface moderna, navegação inferior fixa, cabeçalho responsivo e design centrado no usuário.
